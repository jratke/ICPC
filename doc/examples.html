Example Players

The c++_example and java_example directories contain sample players implemented in C++ and java. The source code for these will demonstrate how a player is to interact with the game, and they may give you some ideas about how your own player could operate. You can compile these players independently and then run them against each other in the game using a command like the following: 

java -jar icypc.jar -player cpp c++_example/camper -player java -cp java_example Hunter

Source code for the java is stored in a java_example subdirectory and is given a matching package name. Teams are reminded that submitted players implemented in Java cannot be organized this way; they must be defined in the default package. The Java example has just been organized this way only to simplify distribution.

C++ Example Player

The C++ example player, camper, tries to build a snow fort near the player's starting corner. Children on the camper team walk around the interior of the fort, creating snowballs from available snow. They opportunistically throw the snowballs at children on the opposing team and pile the snowballs onto the fort wall when no targets are available.

  • Const.h : Game-specific constants used by the C++ player.
  • camper.cpp : Main file for the C++ player.

Java Example Players

Two Java example players are provided. The hunter player concentrates on hitting the opponent with snowballs. Children roam around the field randomly, making snowballs and throwing them at children on the opposing team. The java Planter player concentrates on building snowmen. Children on the planter team constantly try to build new snowman at a location that's far from any existing snowman.

  • Const.java : Game-specific constants used by both of the java players.
  • Hunter.java : Main file for the java Hunter player.
  • Planter.java : Main file for the java Planter player.

C# Example Player

The C# example player is an adaptation of the C++ camper player. Children on the camper team try to build a snow fort in the corner of the field and throw snowballs at the opposing team when they get the chance.

  • Camper.cs : Main file for the C# camper implementation.

Python Example Player

The python example player is an adaptation of the java Hunter player. Children on the hunter team roam around the field randomly, making snowballs and throwing them at children on the opposing team.

  • hunter.py : Main file for the python player.
